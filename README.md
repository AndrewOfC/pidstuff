
# Overview

Simple UI demo of PID controllers at work

# Setup

```bash
$ virtualenv -p python3 pidenv
$ source pidenv/bin/activate
$ pip install -r requirements.txt
$ # do stuff
$ deactivate # when done
```

|      |     |
|-----|-------------------------------------------|
| File|Purpose                                    |
